1.	Firstly you download all the given files.
2.	Install the latest version of jdk on your pc.
3.	Open the project in eclipse or Net Beans.
4.	Click on the run button and it will show you the window.
5.	Then Click on the start scheduler it will show the main window.
6.	Specify the numbers of the processes by clicking create processes.
7.	Then click the start processes button to create the processes’ scheduler.
8.	After done the processes, click on the show result button, it will show you the result of given parameter.
9.	Then click graph it button and it will create all the five parameter’s graph.

=> Note: If you want to run it on cmd, firstly set the path to the bin directory in jdk, and then run the source code
	named "CPU.java";
